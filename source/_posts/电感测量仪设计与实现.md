# 2. 实验原理

&emsp;&emsp;整个振荡器上电之后，电感$L_1$的电流并未发生大的变化，随着电流的不断增大，电阻$R_1$的压降也不断增大，引脚2（TR）和（TH）的输入电压在减小，当2、6引脚上的电压小于VCC/3时电路翻转；3（OUT）引脚输出高电平，引脚7为高阻态（与地断开）。$R_1$和$L_1$组成的回路断开，电感处于放电过程，$R_1$的压降随电流减小而减小，而引脚2（TR）和6（TH）的输入电压不断增大，当电压大于VCC/3时电路再次翻转；3（OUT）引脚输出低电平，引脚7为地阻态（对地导通），$L_1$则再次充电；如此$L_1$重复充电放电过程实现自振荡，根据电路输出波形的高低电平时间可得到其输出频率，公式如下：

&emsp;&emsp;输出波形高电平时间：
$$
t_1=\tau ln\frac{\frac{2}{3}V_{cc}}{\frac{1}{3}V_{cc}}=\tau ln2
$$
&emsp;&emsp;输出波形低电平时间：
$$
t_2=\tau ln\frac{V_{cc}-\frac{1}{3}V_{cc}}{V_{cc}-\frac{2}{3}V_{cc}}=\tau ln2
$$
&emsp;&emsp;输出频率：
$$
f=\frac{1}{t_1+t_2}=\frac{1}{2\tau ln2}=\frac{R_1}{2ln2L_1}
$$
&emsp;&emsp;由上式可得：
$$
L_X=L_1=\frac{R_1}{2ln2f}
$$
其中$\tau =\frac{L_1}{R_1}$
&emsp;&emsp;因此将待测电感作为$L_1$接入到电路中即可测量出待测电感$L_X$的值，电感测量电路中消噪电容$C_1= 0.1uF$，$R_1=100Ω$。电感测量电路构成的多谐振荡器输出引脚3（OUT）的频率较高，因此需要先将输出频率经过74LS90组成的分频电路进行分频，分频之后的频率再通过Arduino采集。74LS90是一个中规模集成的TTL电平计数器，由一个二进制计数器和一个五进制计数器构成，通过引脚间的连接即可实现二分频、五分频以及十分频等功能。

## 3.2 公式原理

frequency = 1.E6/(Pulse_h+Pulse_l)：Pulse函数分别采集输入信号的高位时间与低位时间，二者之和为一个周期，单位为微秒，处理成秒*1.E6

frequency *= Delay : 计算出分频前的频率

inductance = (R* 1000)/(2*frequency*log(2)) : 由频率与电感之前的关系导出

## 3.3 最终程序

```c++
//申明引脚、宏、常量
\#define freqIn 2  //频率引脚
\#define Delay 10   //频率分频值（根据分频电路确定）
const double R = 100.0 ;
//申明频率和电感变量
double frequency, inductance;
void setup()
{
 Serial.begin(9600);
 pinMode(freqIn, INPUT);  //初始化引脚为输入模式
}
void loop()
{
 //使用pulseIn函数读取脉冲高位/低位时间，计时范围为0.01s，输出单位为微妙
 double Pulse_h = pulseIn(freqIn, HIGH, 10000); 
 double Pulse_l = pulseIn(freqIn, LOW, 10000); 
 frequency = 1.E6 / (Pulse_h+Pulse_l); 
 delay(20);
 frequency *= Delay;  //分频之前的频率
 inductance = (R*1000)/(2*frequency*(log(2)));  //电感与频率之间的关系式
 Serial.print("frequency:");  //串口显示
 Serial.print(frequency);
 Serial.println("Hz   ");
 Serial.print("inductance:");
 Serial.print(inductance);
 Serial.println("mH   "); 
 delay(1000);
}
```

# 5. 实验总结

&emsp;&emsp;本次电感测量仪设计与实现项目过程中：

## 5.1 设计步骤

（1）按图接线，取$U_{cc}=5V,R_1=100\Omega,C_1=0.1\mu F$，$L_1$选用配发的电感（222）。观察输出信号的波形、频率。

（2）根据输出信号频率利用74LS90设计相应的分频电路用于Arduino信号采集。

（3）将匹配的输出信号接入Arduino的输入引脚，利用Arduino的高低电平采集函数pulseIn()或外部中断函数进行软件编程，实现信号频率的测量。

（4）依据测量原理中所得出的电感与频率之间的函数关系转换为电感测量，Arduino的串口监视器实时观测电感大小的值。

## 5.2 过程中遇到的问题

（1）使用100欧姆电阻配合所选电感（103）无法输出信号，不振荡，电路连接没问题。

（2）不清楚如何使用74LS90实现10分频的原理。

（3）Aruino如何完整的采集输入信号的一个周期。

（4）串口监视器电感显示结果为0.00。

## 5.3 解决方法

（1）更换一个电感（222），可以使用100欧姆电阻完成信号的输出，频率值为33K。

（2）徒手画出10个脉冲对应的74LS90的Q3端输出波形，发现其周期比输入信号大10倍，也即十分频。

（3）探究Pulse函数的原理，可以采集一个周期HIGH或LOW状态的时间，将两个状态对应的时间加起来即为一个周期的时间。

（4）将代码中关于电感的表达式再乘以1000，使得千分位小数在个位数上显示，解决了显示数据溢出的问题。
